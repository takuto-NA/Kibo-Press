# Kibo-Press ロードマップ

## MVP（0.1.x）

- Pandoc → Typst → PDF パイプライン
- テーマ `business-report`
- コマンド: Export PDF、Export PDF with Theme、Open Preview（既定ビューア）、Create Document Config（front matter 雛形）
- `kiboPress.*` 主要設定と Output Channel
- 結合テスト（Pandoc/Typst 利用可能な環境）

## v1

- 複数テーマ同梱（proposal、manual など）
- 進捗表示の強化
- 画像キャプション、改ページの記法整理
- 必要に応じて簡易プレビュー UI

## v2 以降

- ライブプレビュー、テーマエディタ、プロジェクト単位設定
- DOCX / HTML 出力、複数 Markdown の結合、図表番号・相互参照 など
